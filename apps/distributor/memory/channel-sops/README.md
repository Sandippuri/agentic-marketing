# Channel SOPs

Per-channel guidelines the Content sub-agent reads when drafting for that
channel.

- `linkedin.md` — character limits, hashtag etiquette, when to mention the
  company page, image aspect ratios.
- `x.md` — thread-vs-single guidance, how to chain reposts, link placement.
- `email.md` — subject-line rules, send-time heuristics, link tracking
  conventions.
- `internal-blog.md` — heading style, OG image expectations, syndication card
  copy.
